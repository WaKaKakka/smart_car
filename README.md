# STM32 智能小车项目

基于 STM32F103ZET6 的电机控制小车，实现编码器测速、PID闭环调速和OLED显示。

---

## 硬件架构

### 核心芯片

STM32F103ZET6（Cortex-M3，72MHz，无FPU）

### 外设连接

```
┌─────────────┐
│  STM32F103  │
│             │
│  TIM1_CH1 ─┼──→ TB6612 ──→ 直流电机
│  (PWM输出)   │     ↑
│             │   PA4(AIN1) 方向控制
│  TIM2_CH1 ─┼──← 编码器A相    PA5(AIN2)
│  TIM2_CH2 ─┼──← 编码器B相
│  (编码器接口) │
│             │
│  I2C1_SCL ─┼──→ OLED SSD1306
│  I2C1_SDA ─┼──↔ (128×64)
│             │
│  TIM3 ─────┼──→ 1秒定时中断（更新显示）
└─────────────┘
```

### 时钟配置

- HSE 8MHz → PLL ×9 = 72MHz SYSCLK
- APB1: 36MHz（定时器时钟自动翻倍为72MHz）
- APB2: 72MHz

## 模块原理

### 1. 电机驱动（TB6612）

TB6612是双H桥电机驱动芯片，通过两个GPIO控制方向，一个PWM控制速度。

**引脚定义：**
- AIN1/AIN2：方向控制
- TIM1_CH1（PE9）：PWM输出

**转向逻辑：**

| AIN1 | AIN2 | 状态 |
|------|------|------|
| HIGH | LOW  | 正转 |
| LOW  | HIGH | 反转 |
| LOW  | LOW  | 停止 |

**PWM配置：**
- TIM1周期：1000（0~999）
- 预分频：71（72MHz / 72 = 1MHz计数频率）
- PWM频率：1MHz / 1000 = 1kHz

---

### 2. 编码器测速（TIM2）

使用TIM2的编码器模式（TI12），自动计数A/B相的边沿变化。

**编码器参数：**
- 线数：260线/圈
- 4倍频：260 × 4 = 1040脉冲/圈
- 额定转速：400 RPM
- 空载最大转速：550 RPM

**工作原理：**
- 正转：CNT递增
- 反转：CNT递减
- 每个脉冲对应电机轴的一个固定角度

**读取与转换：**
```c
// 1. 读取并清零（在1秒定时中断中）
pulse_count = (int16_t)__HAL_TIM_GET_COUNTER(&htim2);
__HAL_TIM_SET_COUNTER(&htim2, 0);

// 2. 脉冲数转换为RPM
// 公式：RPM = 脉冲数 / 每圈脉冲数 × 60秒
current_rpm = pulse_count / 1040.0f * 60.0f;
```

- 转为 `int16_t` 解决65535/0溢出问题
- 读取后立即清零，下次读取的就是这段时间内的脉冲数
- 1秒采样周期下，最大可测 32767/1040*60 ≈ 1890 RPM，远超电机极限

---

### 3. OLED显示（SSD1306）

128×64像素，通过I2C通信（Fast模式400kHz）。

**显示流程：**
```c
OLED_NewFrame();                              // 清空显存
OLED_PrintASCIIString(0, 0, "Target: 100");   // 绘制到显存
OLED_PrintASCIIString(0, 20, "Speed: 85");    // 绘制到显存
OLED_ShowFrame();                             // 刷新到屏幕
```

采用**帧缓冲**机制，先在内存中绘制，最后一次性发送到屏幕，避免闪烁。

---

### 4. PID闭环控制

#### 什么是PID

PID根据**误差**（error = 目标值 - 当前值）计算控制输出，让系统趋近目标。

```
误差 e = target - current
```

#### 三个分量

| 分量 | 作用 | 公式 |
|------|------|------|
| **P（比例）** | 反应当下 | Kp × e |
| **I（积分）** | 纠正过去，消除静差 | Ki × ∑e |
| **D（微分）** | 预测未来，减少超调 | Kd × Δe |

- **P**：误差大输出大，响应快，但无法消除稳态误差
- **I**：累积历史误差，消除摩擦等因素导致的静差
- **D**：误差变化快时提前刹车，抑制振荡

#### 本项目使用增量式PID

```c
Δoutput = Kp*(e - e1) + Ki*e + Kd*(e - 2*e1 + e2)
output += Δoutput
```

**优点：**
- 输出增量而非绝对值，天然抗积分饱和
- 不需要额外的积分限幅
- 更适合电机速度控制

#### 保护机制

```c
// 输出限幅（防止PWM超出TIM周期）
if (output > 1000) output = 1000;
if (output < -1000) output = -1000;
```

#### 调参口诀

1. 先调 **Kp**：从小到大，直到电机能响应
2. 再调 **Ki**：消除稳态误差，但不要太大导致超调
3. 最后调 **Kd**：减少超调和振荡

## 主循环流程

```
初始化
  ↓
┌──────────────────────────────────────┐
│  等待 TIM3 中断（1秒）                │
│  ↓                                   │
│  读取编码器脉冲数                     │
│  ↓                                   │
│  脉冲数 → RPM（÷1040×60）            │
│  ↓                                   │
│  PID计算（输入RPM，输出PWM）          │
│  ↓                                   │
│  OLED显示目标转速和实际转速(RPM)      │
│  ↓                                   │
│  回到循环起点                         │
└──────────────────────────────────────┘
```
